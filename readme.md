
## Project Overview
 Secure Authentication Framework that integrates multi-factor authentication and operating system security mechanisms. The system simulates various security modules such as password hashing, buffer overflow protection, privilege escalation prevention, trapdoor detection, and OS security integration through an interactive security simulator dashboard.
This project demonstrates how modern authentication systems work along with operating system security protection mechanisms.
## Features
* Secure User Sign-in (identity enrollment on device)
* Secure Login System (credential check against enrolled identity)
* Multi-Factor Authentication (OTP)
* Password Strength Checker
* Password Hashing Simulation (Argon2)
* Buffer Overflow Protection Simulation
* Privilege Escalation Protection
* Trapdoor Detection
* Operating System Security Integration
* Security Logs Panel
* Terminal-Based Security Simulator
* Progress Bar for Security Modules
* Module Explanation Panel

## Project Workflow
Boot Screen → Sign-in (Enroll Identity) → Login → OTP Verification → Security Simulator Dashboard → OS Security Modules Execution → System Access Granted
Notes:
- Sign-in is required once per browser (stored locally).
- Dashboard/Simulator is blocked until Login + MFA succeeds.
